你或许疑惑我为什么这么说。请允许我细细道来。

我是一名学生，是一个普通的学生。虽然我的考试分数在学校可以占据前列，但是我依旧自认为普通。我出生的家庭或许并非多数人能够拥有的家庭，但是我的家庭依旧算是十四亿中国人中占绝大部分的无产阶级。我的父母由于时代的局限性和个人的经历，拥有各种各样的保守思想。不过，这并不能掩盖他们对我诚恳的教诲。我能生长到今天，他们做的贡献最为主要。当然，我上面讲述的事情可能是这个世界上很多人都有的人生经历，并无出奇。

只不过，作为一个普通的学生，在一个普通的城市，我走入了 Internet。Internet 为我们提供了很多信息和资讯，深刻改变着人们的生活。对于我而言，它让我结识了一些有趣的朋友，让我能够发挥自己的智慧做一些自己想做的事情。

以 Windows 和 macOS 为核心的操作系统为这个世界带来的精彩的互联网体验，但是它们并不尽如人意。使用 Microsoft PowerPoint 能够用图形界面做出操作系统的设计原型，我沉迷于此，或是个人地或是合作地制作了很多 PPT 系统（PPT 制作的设计原型）。它们有 NorthOS，Litmus OS 等等。这是我这几年投入的爱好，但是现在，它们无法占用我整个兴趣爱好的时间。

我热爱探索一切正义的和我所不熟知的东西。既然使用了 Internet，我对它的底层架构感到好奇。我热爱钻研 HTML、CSS 和 JavaScript，并且使用它们构建了一系列网站，并在 Internet 上发现了许多有意义的东西。我使用 Elcetron 技术，与许多志同道合的伙伴一起开发了名叫青韵笔记的给文艺青年的写作软件，并且正在谋划开发更多软件，如纯净的 Windows 应用商店，阅读写作聚合软件等。

但我并不是互联网的忠实用户。在 facebook 更名 Meta，并且掀起了元宇宙的世界潮流之后，我却越发发现了 Internet 的虚拟性给这个世界带来的影响。它使人类沉迷在虚拟的温床中，忽视了对现实世界的探索。我们有了微信，有了 Instegram，有了 Twitter，但是我们得到了什么？我们得到了越来越娱乐化的政治，得到了混乱不堪的互联网不正义。最初为了促进信息交流和人员合作的 Internet 荡然无存，开放自由共享互联网精神已经消亡，一个个信息孤岛正在形成。

反观，从 20 世纪 70 年代最后一次登上月球，距今已经五十多年。人类再也没有登上月球。当年奋勇争先探索宇宙的精神随着虚拟社交的兴起而消灭，人类登上火星和其他星球的理想已经破灭。我们需要探索星空。而在地球上，众多社会问题依旧得不到解决，剥削阶级仍旧在压迫广大的劳动人民。

我不是一个厌世的人，不想像陶渊明一样“归去来兮，田园将芜胡不归”，更不是极端的反人类者像降临派一样希望毁灭人类以消灭人类的罪恶。

我相信人之初性本善，虽然有的人因为生理结构导致与常人不同，但是人毕竟是人，与其他生物不同。我信仰马克思主义，希望消灭私有制达到天下大同。但是，革命的道路千万里，必然充满鲜血与汗水。我从不寄希望于科学手段改变人类意志、清除人类记忆。民主自由平等的确是普世的价值，但是完全不是西方口中的那个样子；为了维护人之所以为人的基本权利，利用技术手段改变人类不可取。

话说回来，我称你为同志，是相信你也有探索星空的理想，也有像保尔柯察金一样解放人类的目标。即使没有，关注我，我能帮你培养。

不过这也仅仅是远大目标。在现实生活中，我们处处受到资本的束缚和资本的代言人——政府的管理。无论是无恶不作、只为资本增殖服务的西方政府，还是打着马克思主义旗号反马克思主义的修正主义某东方大国政府，都控制着人民。

在普普通通的生活里，我是一个现实主义的理想主义文艺青年。我知道“文艺青年”是小布尔乔亚的称谓，但是我却受到罗永浩的影响。我怀念过去慢慢的社会，也向往未来科技化的社会。我幻想科幻小说中的极客未来，也热爱古文古籍中风和日丽的自然。我对手机电脑汽车等现代科技产物感兴趣，也对榫卯建筑等文艺有所研究。或许在这些方面我是割裂的，但是多样性才能成就一个真实的人。的确，我在各个领域都是割裂的。我感觉文理分科是最恶劣的政策，因为我本人就是两边通吃的学生。我会写优美的文学作品，也能写计算机程序。
